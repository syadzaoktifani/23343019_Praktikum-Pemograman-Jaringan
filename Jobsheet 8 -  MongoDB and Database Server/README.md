# Kesimpulan

Dari jobsheet ini, saya memperoleh pemahaman dan pengalaman praktis dalam menggunakan database NoSQL MongoDB sebagai database server pada aplikasi berbasis Node.js. Saya mempelajari perbedaan mendasar antara database relasional (SQL) dan database NoSQL, khususnya dalam hal struktur data, fleksibilitas skema, dan cara penyimpanan data berbasis dokumen.

Melalui latihan ini, saya berhasil melakukan koneksi Node.js ke MongoDB, memahami struktur database, collection, dan document, serta menggunakan ObjectId sebagai identitas unik setiap data. Saya juga mempraktikkan operasi CRUD (Create, Read, Update, Delete) secara langsung menggunakan MongoDB Native Driver, termasuk penggunaan insertOne, insertMany, find, updateOne, updateMany, deleteOne, dan deleteMany.

Selain itu, saya memahami bagaimana MongoDB menangani data secara asinkron, cara membaca dan memodifikasi data berdasarkan kriteria tertentu, serta memverifikasi hasil operasi database menggunakan MongoDB Compass sebagai alat bantu visual.

Dengan menyelesaikan jobsheet ini, saya memperoleh pemahaman menyeluruh tentang peran database server dalam aplikasi backend, serta mampu mengelola data secara efisien menggunakan MongoDB sebagai solusi NoSQL yang fleksibel dan scalable untuk aplikasi web modern.
